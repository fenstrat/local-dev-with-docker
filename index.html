What is Docker

  • Operating-system-level virtualization aka containerization
  • Builds on Linux kernel's cgroups (available in the kernel since Jan 2008)
  • Containers run on multiple platforms, Linux, OSX, Windows

Containers built from a Dockerfile


FROM ubuntu
MAINTAINER Saul Willers

RUN apt-get install -y apache2

EXPOSE 80

ENTRYPOINT ["/usr/sbin/apache2"]
CMD ["-D", "FOREGROUND"]

What about multiple containers?

Docker Compose

  • Tool for defining and running multi-container Docker applications
  • Single command to create and start all the services

Defined in docker-compose.yml


version: '3'
services:
  php:
    image: wodby/php-apache
    # This makes the container run on the same network stack as your
    # workstation. Meaning that you can interact on "localhost".
    network_mode: host
  db:
    image: mariadb
    network_mode: host
    environment:
      - MYSQL_ROOT_PASSWORD=root
      - MYSQL_DATABASE=local
      ...
  mail:
    image: mailhog/mailhog
    network_mode: host

  # Supports any required service:
  varnish|memcache|phantomjs|chromedriver|solr

Control all the containers!

  • docker-compose up
  • docker-compose stop|down
  • docker-compose exec php bash
